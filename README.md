# Jogo RPG Básico em Flutter 🧙‍♂️🎮

Este repositório contém o projeto de um jogo básico de RPG desenvolvido em **Flutter** como parte da disciplina de **Computação Gráfica** na **Universidade UNIVATES**. O jogo implementa mecânicas essenciais de RPG, como **vida**, **movimentação**, **dano**, e **sistema de experiência (XP)**.

## 🛠️ Funcionalidades Implementadas

- **Movimentação**: Controle fluido do personagem em um ambiente 2D.
- **Barra de Vida**: Representação visual da saúde do personagem.
- **Sistema de Dano**: Redução de vida ao receber ataques.
- **Design Responsivo**: Compatível com diferentes tamanhos de tela.

## 📂 Estrutura do Projeto

- `lib/`
  - `main.dart`: Arquivo principal do projeto.
  - `interface/`: Componentes de interface, como barra de vida e botões de controle.
  - `pages/`: Telas do jogo, como menu inicial e gameplay.
  - `player/` `enemies/`: Classes representando personagens, inimigos e atributos do jogo, incluindo cálculos de dano e movimentação.
  
- `assets/`: 
  - Imagens e sprites utilizados no jogo.

## 🚀 Como Executar

Clone este repositório:
  ```bash
  git clone https://github.com/usuario/jogo-rpg-flutter.git
  ```

Instale as dependências:
  ```bash
  flutter pub get
  ```

Execute o projeto:
  ```bash
  flutter run
  ```
